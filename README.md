# Portfolio Starter Kit

A premium, modern portfolio website built with HTML, CSS, and JavaScript.

## 🚀 Quick Start

1.  **Customize Content**: Open `index.html` and replace the placeholder text and images with your own.
2.  **Styles**: Update `style.css` to change colors, fonts, or layout.
3.  **Deploy**: Follow the steps below to host it on GitHub Pages.

## 📦 How to Deploy to GitHub Pages

### Option 1: Using GitHub Desktop / Command Line
1.  Create a new repository on GitHub (e.g., `portfolio`).
2.  Initialize git in this folder:
    ```bash
    git init
    git add .
    git commit -m "initial portfolio commit"
    ```
3.  Connect to your GitHub repo and push:
    ```bash
    git remote add origin https://github.com/YOUR_USERNAME/portfolio.git
    git push -u origin main
    ```
4.  On GitHub, go to **Settings > Pages**.
5.  Under **Build and deployment**, select `Deploy from a branch`.
6.  Set the branch to `main` and the folder to `/ (root)`.
7.  Click **Save**. Your site will be live at `https://YOUR_USERNAME.github.io/portfolio/`.

### Option 2: Uploading via Website
1.  Create a new repository on GitHub.
2.  Click **uploading an existing file**.
3.  Drag and drop all files from this folder (`index.html`, `style.css`, `script.js`).
4.  Commit the changes.
5.  Follow steps 4-7 from Option 1.
